# Webform Backend
Used to interact with Strapi APIs
## Dependencies
Python 3.11
Uvicorn
Mangum
fastAPI
## Running the app
```
pip install [dependencies]
```
then
```
python -m uvicorn main:app --reload
```
will run the app on localhost:8000
## API Docs
After running the app. Go to ```localhost:8000/docs```
